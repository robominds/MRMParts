# MRMParts

A KiCad symbol library containing custom electronic component symbols.

## Contents

This library currently includes:

- **MAX118** - 8-channel, 8-bit analog-to-digital converter (ADC) with multiplexed inputs
  - [Datasheet](https://www.analog.com/media/en/technical-documentation/data-sheets/MAX114-MAX118.pdf)

## Installation

### Option 1: Direct Download
1. Download `MRM332Parts.kicad_sym`
2. In KiCad, go to **Preferences â†’ Manage Symbol Libraries**
3. Add a new library entry pointing to the downloaded `.kicad_sym` file

### Option 2: Git Clone
```bash
git clone https://github.com/yourusername/MRMParts.git
```

Then add the library path in KiCad's symbol library manager.

## Usage

Once installed, the symbols will be available in KiCad's schematic editor through the symbol chooser.

## File Format

- Created with KiCad 9.0
- Symbol library format version: 20241209
